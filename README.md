# WebEng--Assigment2
This .NET 8 Web API manages internet "brain rot." Using a Service Pattern, it tracks BrainRotters and ViralAilments.  Endpoints:  Rotters: Get Lobby, Vibe Check (ID), and Banish (Delete).  Ailments: View Trends, Fanum Tax (Update logic), and Exorcise (Delete).  Architecture includes DI, Interfaces, and RESTful Controllers.
